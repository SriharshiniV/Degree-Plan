Directory Structure  

The project directory is named "Project"
It contains three folders
-Documents
-Source
-UserManual
Documents folder contains five sub folders
-Meeting Minutes : Contains the meeting minutes
-Planning Documents 
-Presentations 
-Project Reports : Contains project deliverable documents
-Test Documentation
Source folder contains three subfolders 
-Project src
-Resources 
-Test Src

The paths are as follows:
	
Documents : Shan\Project\Documents
Error Documentation : Shan\Project\Documents\Error Documentation

Meeting Minutes : Shan\Project\Documents\Meeting Minutes

Planning Documents: Shan\Project\Documents\Planning Documents\

Gantt Charts : Shan\Project\Documents\Planning Documents\Gantt Charts
Pert Charts : Shan\Project\Documents\Planning Documents\Pert Charts
UML Diagrams : Shan\Project\Documents\Planning Documents\UML Diagrams

Presentations : Shan\Project\Documents\Presentations
Project Reports : Shan\Project\Documents\Project Reports

Source : Shan\Project\Source
Project Src : Shan\Project\Source\Project Src
Resources : Shan\Project\Source\Resources
Test Src : Shan\Project\Source\Test Src

User Manual : Shan\Project\User Manual
